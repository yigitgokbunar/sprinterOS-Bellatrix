# sprinterOS-Bellatrix
A sprinterOS version designed for RPI2B and RPI Zero 2W.
![alt text][logo]

[logo]: https://github.com/yigitgokbunar/sprinterOS-Bellatrix/blob/main/sprinterOSlogobelatrix.png


Bellatrix is the first sprinterOS that comes with a very lightweight and efficient desktop environment: iceWM.  A timeless classic. Thanks to its efficiency, sprinterOS Bellatrix only needs 256MB of RAM to work stably. It includes lightweight software, so there are no web-based slop programs, and there is no unnecessary performance loss. I designed it with "Be useful first" motto.

To make appliance.kiwi to iso or a raw, you can read: https://osinside.github.io/kiwi/building_images.html
